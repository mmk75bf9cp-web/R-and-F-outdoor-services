## How to Use

### Opening the Website
1. Open `index.html` in any web browser
2. The website will load with all features active

### Submitting Reviews
1. Scroll to the "Reviews" section
2. Fill in your name, email, and rating
3. Write your review text
4. Click "Submit Review"
5. Your review will appear at the top of the reviews grid

### Requesting a Consultation
1. Scroll to "Get Started Today" section (or click the CTA buttons)
2. Fill in your name, email, phone, and project details
3. Click "Request Consultation"
4. You'll receive a confirmation message

## Customization

You can easily customize:
- **Contact Information** - Update phone, email, address, and hours in the contact section
- **Service Details** - Edit service descriptions in the Services grid
- **Colors** - Modify CSS variables in `styles.css` (`:root` section)
- **Add More Reviews** - Duplicate review cards in the HTML
- **Social Media Links** - Update social links in the footer
- **Business Hours** - Change hours in the contact section
